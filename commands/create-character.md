Invoke the `create-character` skill from the mythoi plugin and walk the user through City of Mist character creation step by step. Stay conversational and use the bundled MCP tools (`list_themebooks`, `get_themebook`, `themebook_question`, `validate_character`, `save_character`, `render_sheet`) to keep all data lookups deterministic.

Begin by asking for the user's Mythos concept, Logos concept, awareness level, and a name.
